# valentin-surprise

Un detalle romántico (HTML + CSS + JavaScript) para sorprender a tu novia en San Valentín.

## Abrir
- Solo abre `index.html` en tu navegador.

## Música (opcional)
1. Crea la carpeta `audio/`
2. Coloca un mp3, por ejemplo: `audio/humbe.mp3`
3. En `index.html` descomenta la línea del `<source>`

> Nota: por políticas del navegador, la música solo se puede reproducir después de un clic/tap.
